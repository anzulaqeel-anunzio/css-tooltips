# Pure CSS Tooltip Library

A lightweight, dependency-free tooltip library using only CSS data attributes.

## Features

-   **Zero JavaScript**: Uses `::before` and `::after` pseudo-elements.
-   **Data Attributes**: Define content via `data-tooltip`.
-   **Positions**: Top, Bottom, Left, Right.

## Installation

1.  **Clone the repository** OR copy the CSS.
2.  **HTML Usage**:
    ```html
    <link rel="stylesheet" href="css/tooltips.css">
    ```

## Usage

Add `data-tooltip` to any element:

```html
<button data-tooltip="This is a helper text">Hover me</button>
```

Change position with modifiers:
```html
<button data-tooltip="I'm on top!" class="tooltip-top">Top</button>
<button data-tooltip="I'm on right!" class="tooltip-right">Right</button>
```
